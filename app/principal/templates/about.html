Aplicativo Modelo

Aplicativo que servirá de modelo para o desenvolvimento de novos aplicativo web utilizando o framework Flask!



Passos para a inicialização:


  1. Instalar o Pipenv: pip install pipenv
  2. Criar uma nova pasta: md nova_pasta
  3. Copiar o conteúdo do projeto 'framework' para a nova_pasta
  4. Entrar na nova pasta: cd nova_pasta
  5. Criar uma ambiente virtual para o projeto: pipenv install
  6. Acessar o ambiente virtual: pipenv shell
  7. Instalar os pacotes do projeto:
    • pipenv install flask
    • pipenv install flask-wtf
    • pipenv install flask-sqlalchemy
    • pipenv install flask-bcrypt
    • pipenv install flask-login
    • pipenv install Pillow
    • pipenv install flask-mail
    • pipenv install mysqlclient-1.4.2-cp37-cp37m-win_amd64.whl
    • pipenv install reportlab
  8. Cria um novo Schema no Mysql com o nome do projeto:
    • DROP SCHEMA IF EXISTS `db_nome_do_projeto` ;
    • CREATE SCHEMA IF NOT EXISTS `db_nome_do_projeto` DEFAULT CHARACTER SET utf8 ;
  9. Criar as tabelas no Schema `db_nome_do_projeto`:
    • Editar o programa: create_db.py
    • Ajustar o parâmetro: 'SQLALCHEMY_DATABASE_URI'
    • Executar o programa: python create_db.py
  10. Entrar na pasta app: cd app
  11. Ajustar as configuração do banco de dados no arquivo config.py
  12. Executar o aplicativo: python run.py
  13. Acessar o aplicativo no browser pelo endereço: http://127.0.0.1:5000
